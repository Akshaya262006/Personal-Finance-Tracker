Personal Finance Tracker

Project Overview
Personal Finance Tracker is a simple web application that helps users manage their income and expenses efficiently. The application allows users to add, view, and manage financial transactions while tracking their overall balance and financial health.

 Features

 Dashboard
- View Total Income
- View Total Expenses
- View Current Balance
- Financial Health Indicator

Income Management
- Add income sources
- Store income records
- View income history
- Delete income records

Expense Management
- Add expenses with category
- View expense transactions
- Manage expense records
- Delete expenses

Theme Feature
- Toggle between Light Mode and Dark Mode
- User-friendly interface

Financial Health Card
- Calculates financial health percentage based on income and expenses
- Gives a quick overview of spending habits

 Technologies Used
- HTML
- CSS
- JavaScript

How to Run the Project

1. Clone the repository
bash
git clone https://github.com/Akshaya262006/Personal-Finance-Tracker.git

2. Open the project folder

bash
cd Personal-Finance-Tracker

3. Open `index.html` in your browser

OR

4. Run using VS Code Live Server

Financial Health Formula

Financial Health (%) =
(Balance / Total Income) × 100

Where:

Balance = Total Income - Total Expense
